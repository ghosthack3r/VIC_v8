# OMNI VIC Agent

This is the core system pack for VIC (Vehicle Intelligence & Command).

## Purpose
Provides personality, behavior logic, state rules, response rules, and voice guidance for VIC.

## Usage
1. Load `system-prompt.md` into the AI agent.
2. Use `state-engine.json`, `personality-config.json`, and `event-triggers.json` for runtime behavior.
3. Connect VIC to simulated or real vehicle telemetry.
4. Integrate with the OMNI ecosystem and command center.

## Recommended First Build
Ask Codex or your dev agent to:

> Build a VIC agent using this system pack. Implement the state engine, personality filter, and event trigger system. Connect it to a simple UI and simulated driving data.

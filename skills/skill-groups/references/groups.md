## Toggle Notes

- Safe toggle candidates are primarily personal skills in `C:\Users\ghost\.agents\skills`.
- Local Codex skills in `C:\Users\ghost\.codex\skills` may be personal, but do not move `.system` skills.
- Plugin skills are controlled by plugin installation/enabling, not by moving files from `.codex\plugins\cache`.
- Disabling any Superpowers process skill can weaken quality controls. Require explicit user intent.
- After toggling, start a fresh Codex session or reload skills to confirm the new active set.
